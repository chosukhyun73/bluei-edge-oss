# 문제 해결

흔한 증상과 확인할 사항입니다. 설치/배포 수준의 문제는 이 운영자 가이드가 아닌 내부
배포 가이드를 사용하세요.

| 증상 | 추정 원인 | 조치 |
|---------|--------------|------------|
| 대시보드가 비어 있음 / 로드되지 않음 | 런타임 미실행 또는 잘못된 주소 | `http://<edge-ip>:8080/` 주소에 접속했는지, GX10 / 런타임이 켜져 있는지 확인하세요. |
| 브라우저 콘솔에 `401 Unauthorized` | 운영자 인증 불일치 | 지원되는 진입점에서 대시보드를 다시 여세요. 계속되면 백엔드 인증/토큰을 점검해야 합니다. |
| 센서가 오래된 값 또는 `—`를 표시 | 센서가 임계치를 넘겨 오프라인, 배선 문제, 또는 수집기 중단 | 센서 연결과 수집기 상태를 확인하세요. 새 데이터가 들어오면 표시가 정상화됩니다. |
| 카메라가 연결되지 않음 | 잘못된 RTSP 주소 또는 비밀번호 | 해당 카메라의 RTSP 정보를 다시 확인하고 비밀번호를 다시 저장하세요. |
| 급이 사이클이 차단됨 | 안전 게이트(용존산소(DO) / 수온 / 센서) | 게이트의 사유를 읽고 수질 상태를 해결한 뒤 다시 시도하세요. [일상 운영 → 안전 게이트](03-daily-operations.md)를 참고하세요. |
| 시작 시 "이미 진행 중인 사이클이 있습니다" | 수조에 이미 실행 중인 사이클이 있음 | 진행 중인 사이클을 먼저 중지/중단한 뒤 새 사이클을 시작하세요. |
| 컨트롤러가 감지되지 않음 | USB / ESP32가 인식되지 않음 | 기기를 다시 연결하고 **AI 관리 → 컨트롤러**에서 자동 등록을 다시 시도하세요. |
| 밀도가 `0.0`으로 표시 / 성장 단계가 "미정" | 수조 용적 또는 입식이 설정되지 않음 | 수조 물리 정보와 입식을 완료하세요 — [초기 설정](02-initial-setup.md)을 참고하세요. |
| UI 텍스트가 잘못된 언어로 표시 | 언어 토글 | 헤더에서 **English / 한국어**를 전환하세요. 운영자가 입력한 이름은 번역되지 않습니다(데이터이기 때문). |

> ℹ️ 기기 수준의 하드웨어 또는 네트워크 결함(전원, 케이블, GX10 자체)으로 보이는
> 문제라면 현장 지원으로 에스컬레이션하세요. 이는 대시보드 영역 밖입니다.
