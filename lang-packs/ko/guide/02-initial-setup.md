# 초기 설정 (최초 1회)

사이트를 처음 구축할 때 한 번만 수행합니다. 순서가 중요합니다. 각 단계는 이전 단계를
바탕으로 진행됩니다(수조는 그룹에 속하고, 센서는 수조에 매핑되며, 입식은 수조를 채웁니다).

> ℹ️ 여기서 입력하는 이름(농장명, 사이트명, 그룹명)은 **사용자 데이터**이며 두 언어 모두에서
> 입력한 그대로 표시됩니다. 시스템 라벨만 번역됩니다.

## 1. 농장과 사이트 등록

**농장**은 법적 주체(사업자/법인)이고, **사이트(양식장)**는 면허가 부여된 양식장 위치입니다.

1. 선택기 행에서 농장 **① + 신규**와 사이트 **② + 신규** 버튼을 사용합니다.

![농장 및 사이트 선택기 행, 각각 + 신규 버튼 포함](../images/SS-03-farm-site-selector.png)

2. **신규 농장 등록** — 법인/사업자명과 등록 번호를 입력합니다. 시스템 ID는 자동으로
   생성됩니다.

![신규 농장 등록 모달 — 법인명, 등록 번호, 인증](../images/SS-04-new-farm.png)

3. **신규 사이트 등록** — 어업 면허명과 번호를 입력하고, 사이트 유형(**육상(RAS)** 또는
   **해상(가두리)**)을 선택하며, 선택적으로 주소를 입력합니다.

![신규 사이트 등록 모달 — 어업 면허, 사이트 유형(육상/해상), 주소](../images/SS-05-new-site.png)

> ℹ️ 잘못 입력한 경우, 올바른 항목을 등록하고 잘못된 항목을 삭제하세요. 이름은 직접 수정할
> 수 없습니다.

## 2. 그룹 생성

그룹은 수조를 묶어 관리합니다(예: 하나의 RAS 라인). **그룹** 패널을 열고 **신규 그룹**을
선택한 뒤 ID, 이름, 색상을 설정합니다.

![신규 그룹 등록 모달 — 그룹 ID, 이름, 설명, 색상](../images/SS-06-new-group.png)

## 3. 수조 및 물리 정보 등록

각 가두리(케이지)/수조와 물리적 세부 정보를 추가합니다. **형태**(원형 / 정사각형 /
직사각형)와 치수를 입력하면 **용적(m³)**이 자동으로 계산되며, 이후 입식 밀도 산정에
사용됩니다.

![그룹의 가두리/수조 탭 아래 수조 추가 양식 — ID, 종, 시스템 유형, 용적 자동 계산을 포함한 물리 정보](../images/SS-07-tank-create.png)

## 4. 센서, 장비, 카메라 매핑

수조 설정에서 각 수조에 속하는 하드웨어를 연결합니다.

- **센서** — 수온, 용존산소(DO), pH, 염분 등.
- **장비(액추에이터)** — 사료기, 펌프, 히터, UV 등.
- **카메라** — RTSP 연결 정보 포함(비밀번호는 지금 저장하거나 나중에 추가할 수 있습니다).

각 행에 추가 진입점이 있으며, 매핑은 저장하면 수조 목록에 표시됩니다.

> 📸 **SS-08** · 센서 / 액추에이터 / 카메라 매핑 · _캡처 예정_ — [레지스트리](../SCREENSHOTS.md#ss-08) 참고

## 5. 컨트롤러(ESP32) 등록

컨트롤러는 장비를 구동합니다. **인공지능 관리 (AI) → 컨트롤러**에서 연결된 ESP32를
**USB로 자동 등록**할 수 있습니다. 패널이 장치를 감지하여 등록 과정을 안내합니다.

> 📸 **SS-09** · 컨트롤러 USB 자동 등록 · _캡처 예정_ — [레지스트리](../SCREENSHOTS.md#ss-09) 참고

## 6. 입식 등록

마지막으로 각 수조에 입식한 내용(종, 수량, 평균 중량)을 기록합니다. 이렇게 하면 빈 수조가
활성 수조로 전환되어 사료 사이클과 밀도 점검이 가능해집니다.

> 📸 **SS-10** · 입식 등록 · _캡처 예정_ — [레지스트리](../SCREENSHOTS.md#ss-10) 참고

> ✅ 입식 후 **사이트 수조 관리**를 열면 수조 카드가 활성화되고 사료 사이클을 시작할 수
> 있습니다. 계속해서 **일상 운영**으로 진행하세요.
