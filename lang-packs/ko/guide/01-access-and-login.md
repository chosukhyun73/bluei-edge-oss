# 접속 및 로그인

## 대시보드 열기

GX10과 **같은 네트워크**에 있는 기기에서 브라우저를 열고 다음 주소로 접속합니다.

```
http://<edge-ip>:8080/
```

`<edge-ip>`를 LAN상의 GX10 IP 주소로 바꿉니다. 대시보드와 API가 같은 주소에서 제공되므로
브라우저 쪽에 추가 설정이 필요하지 않습니다.

> ℹ️ GX10에 데스크톱 환경이 있는 경우, 대시보드가 자체 창에서 자동으로 열릴 수도 있습니다.
> 이 경우 별도의 브라우저를 사용할 필요가 없습니다.

## 휴대폰 승인으로 로그인

로그인은 **이 GX10**을 **bluei 계정**과 연결하는 과정이며, 휴대폰에서 승인합니다.

1. 로그인 화면에서 **① bluei 계정으로 로그인**을 선택합니다.
2. 대시보드에 **② 승인 번호**와 "승인 대기 중…"이 표시됩니다.
3. **휴대폰의 bluei 앱**을 열어 일치하는 번호를 찾고 **승인**합니다.
4. 승인되면 화면에 **연결됨**이 표시되고 대시보드가 로드됩니다.

![로그인 화면 — GX10을 bluei 계정에 연결한 뒤 휴대폰에서 승인](../images/SS-01-login.png)

승인이 **거부**되거나 요청이 **만료**되면 화면에 상황이 설명되며, **다시 시도**를 선택해
재시작합니다. 네트워크 문제가 있으면 "로그인 요청에 실패했습니다. 네트워크를 확인해
주세요."가 표시됩니다.

## 화면 구성 이해

로그인 후, 상단에 항상 표시되는 **헤더 바**가 제어 영역입니다.

- **① 시스템 상태** — `SYSTEM ONLINE`은 백엔드가 정상임을 나타냅니다.
- **② 언어** — **English** / **한국어**를 전환하며, 해당 브라우저에 기억됩니다.
- **③ AI 운영 도우미** — 운영 도움말 패널을 엽니다(*시스템 운영* 참고).
- **④ 안전 종료** — GX10 **전원을 끄기 직전에** 백엔드를 안전하게 중지합니다
  (*시스템 운영* 참고 — 함부로 사용하지 마세요).

![헤더 바 — 시스템 상태, 언어 선택기, AI 운영 도우미, 안전 종료](../images/SS-02-header.png)

헤더 아래에는 **농장 / 사이트 선택기**와 네 개의 주요 탭이 있습니다.

| 탭 | 용도 |
|-----|---------|
| **사이트 수조 관리** | 그룹·수조 상태, 생산 개요, 사료 사이클 |
| **인공지능 관리 (AI)** | 운영 정책, 추론, 현장 학습, 학습 기반 안전, 컨트롤러, 관리 |
| **생산·이벤트 기록** | 폐사, 처치, 이동 등의 기록 |
| **입식·출하·거래처** | 입식, 출하, 거래처, 거래 서류 |

> ℹ️ 대시보드를 처음 열면 대부분의 패널이 비어 있는데, 정상입니다.
> 계속해서 **초기 설정**으로 농장, 사이트, 수조를 등록하세요.
