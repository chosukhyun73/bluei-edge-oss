# 시스템 운영

일상적인 시스템 수준 제어는 헤더에 있습니다. **AI 운영 도우미**, 언어 선택기,
그리고 **안전 종료**입니다.

## AI 운영 도우미

헤더의 **AI 운영 도우미** 버튼으로 엽니다. 시스템 동작 방식과 현재 수조/시스템
상태를 물어볼 때 사용하며, 시작을 돕는 예시 프롬프트가 제공됩니다.

> ℹ️ 이 도우미는 **참고용**입니다. 시스템을 이해하고 탐색하는 것을 도와줍니다.
> 실시간 안전이나 급이 제어 결정은 **하지 않으며**, 그러한 결정은 시스템(규칙 엔진 /
> 중재자(Arbiter))이 담당합니다. [개요 및 안전 원칙](00-overview.md)을 참고하세요.

![예시 프롬프트와 질문 입력란이 있는 AI 운영 도우미 패널](../images/SS-20-ai-assistant.png)

## 안전 종료

> ⚠️ **안전 종료는 GX10 전원을 끄기 직전에만** 사용하세요. 수집, 급이, 안전 모니터링,
> 동기화를 **전부** 중단합니다. 종료 전에 데이터는 안전하게 저장됩니다.

1. 헤더 → **안전 종료**.
2. 대화 상자에서 확인합니다. 종료가 진행되는 동안 진행 상황이 표시됩니다.
3. "종료 완료 — 전원을 꺼도 됩니다"라고 표시되면 전원을 차단해도 안전합니다.
4. 나중에 다시 시작하려면 대시보드/런타임을 재실행합니다(데스크톱이 있는 GX10의 경우
   바탕화면 아이콘).

![수집, 급이, 안전, 동기화가 중단됨을 경고하는 안전 종료 확인 대화 상자](../images/SS-21-safe-shutdown.png)

## 오프라인 동작 및 동기화

bluei-edge는 **오프라인 지원**입니다. 인터넷 연결 없이도 수집, 기록, 안전 로직 적용을
계속합니다. 이벤트는 먼저 로컬에 저장되며, 연결이 복구되면 `app.bluei.kr`로 동기화됩니다.

- 헤더의 **SYSTEM ONLINE**은 백엔드가 정상임을 나타냅니다(이는 인터넷 상태가 아니라
  로컬 런타임 상태입니다).
- 동기화에는 운영자의 별도 조치가 필요 없으며, 네트워크가 복구되면 자동으로 재개됩니다.

## 언어

헤더의 언어 선택기로 **English / 한국어**를 전환합니다. 선택은 해당 브라우저에
기억됩니다. 운영자가 입력한 이름(농장, 사이트, 그룹, 수조)은 입력한 그대로 유지되며,
시스템 라벨만 번역됩니다.
